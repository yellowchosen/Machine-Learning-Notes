# Cost Function
In multiple linear regression, its cost function should be very similiar with simple linear regression:
$$J(w_0, w_1,..., w_n,b) = \frac{1}{2m}\sum_{i=1}^{m}(f_{w,b}(x^{(i)}) - y^{(i)})$$ and we define $f_{w, b}(x) = w^Tx$
